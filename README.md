### Compilation: 
> Go inside the Makefiles Directory and then for each of the HT or HP compile with: 

```bash
$ make
```

### Execution:
> Go inside the Makefiles Directory and then for each of the HT or HP run with:

```bash
$ make run  
```

To delete binary files: 
```
$ make clean
```
